# Story 009.006: Exportação Excel e PDF do Plano PMP

## Status
Draft

## Story

**Como** Gestor PCP ou Diretor,
**Eu quero** exportar o plano de produção para Excel e PDF,
**Para que** eu possa apresentar e discutir os dados na reunião semanal de revisão de sexta-feira.

## Acceptance Criteria

1. Botão "Exportar" disponível tanto na tela de edição do plano (`/pmp/:pmp_id`) quanto no dashboard (`/pmp/dashboard`)
2. Menu de exportação oferece duas opções: "Excel (.xlsx)" e "PDF"
3. Seleção de escopo antes de exportar: "Mês Completo" ou "Semana [1-5]"
4. **Excel** — aba "Plano de Produção":
   - Cabeçalho: Plano Mestre de Produção — [Mês por Extenso] / [Ano]
   - Colunas: Linha, SKU Código, SKU Descrição, Semana, Data Início, Data Fim, Qtd Planejada, Horas Planejadas
   - Formatação pt-BR (decimal vírgula, milhar ponto)
   - Linha de totais ao final: Total Qtd Planejada, Total Horas Planejadas
5. **Excel** — aba "Headcount":
   - Cabeçalho: Headcount por Linha — [Mês/Ano]
   - Colunas: Linha, Lotação, Cargo, HC Desejado, HC Atual, Gap
   - Sinalização de Gap: célula vermelha se gap < 0
6. **Excel** — aba "Comparativo" (apenas quando exportado do dashboard):
   - Colunas: Linha, SKU, Semana, Qtd Planejada, Qtd Realizada, % Aderência, Horas Planejadas, Horas Realizadas
7. **PDF**:
   - Cabeçalho: "Plano Mestre de Produção — [Mês por Extenso] / [Ano]"
   - Data de geração: "Gerado em: [DD/MM/AAAA HH:mm]"
   - Tabela de itens com layout limpo (sem cores excessivas)
   - Seção de Headcount ao final
   - Rodapé com número de página
8. Nome do arquivo exportado: `PMP_[Mes]_[Ano]_[DataGeracao].xlsx` ou `.pdf`
9. Usar bibliotecas padrão do projeto: XLSX para Excel, jsPDF + html2canvas para PDF
10. Exportação disponível para `perfil != 'Operador'`

### Estrutura do Excel (XLSX)
```typescript
import * as XLSX from 'xlsx';

const wb = XLSX.utils.book_new();

// Aba 1: Plano de Produção
const wsPlano = XLSX.utils.aoa_to_sheet([
  ['Plano Mestre de Produção — Março / 2026'],
  [],
  ['Linha', 'SKU Código', 'SKU Descrição', 'Semana', 'Data Início', 'Data Fim', 'Qtd Planejada', 'Horas Planejadas'],
  ...itens.map(i => [
    i.linha_nome,
    i.produto_referencia,
    i.produto_descricao,
    `Semana ${i.semana}`,
    formatarData(i.dt_inicio),
    formatarData(i.dt_fim),
    // IMPORTANTE: string formatada pt-BR para evitar interpretação errada pelo Excel
    i.qtd_planejada.toLocaleString('pt-BR'),
    i.horas_planejadas?.toLocaleString('pt-BR', { minimumFractionDigits: 2 }) ?? '—',
  ]),
  // Linha de totais
  ['', '', '', '', '', 'TOTAL',
    totalQtd.toLocaleString('pt-BR'),
    totalHoras.toLocaleString('pt-BR', { minimumFractionDigits: 2 }),
  ],
]);

XLSX.utils.book_append_sheet(wb, wsPlano, 'Plano de Produção');

// Exportar
XLSX.writeFile(wb, `PMP_Marco_2026_${dataHoje}.xlsx`);
```

### Formatação de Data pt-BR
```typescript
const formatarData = (dateStr: string) => {
  const [ano, mes, dia] = dateStr.split('-');
  return `${dia}/${mes}/${ano}`;
};
```

# Story 009.002: API TypeScript — PMP

## Status
Draft

## Story

**Como** desenvolvedor do sistema OEE,
**Eu quero** criar a camada de serviços TypeScript para o módulo PMP,
**Para que** o frontend consuma funções tipadas com regras de negócio centralizadas (cálculo de horas planejadas e headcount atual).

## Acceptance Criteria

1. Arquivo `src/types/pmp.ts` criado com todos os tipos TypeScript do módulo
2. Arquivo `src/services/api/pmp.api.ts` criado com funções de CRUD para plano, itens e headcount
3. Ao salvar `tbpmp_item`, `horas_planejadas` é calculado: busca `velocidade` em `tbvelocidadenominal` WHERE `linhaproducao_id` E `produto_id` E `deletado = 'N'`; calcula `qtd_planejada / velocidade`
4. Retorna aviso (`warning`) quando velocidade nominal não está cadastrada para o par linha/SKU
5. Ao consultar headcount, `headcount_atual` é calculado via: `COUNT(tbfuncionario WHERE lotacao_id AND cargo_id AND ativo = true AND dt_rescisao IS NULL)`
6. Função `getPmpComparativo` retorna dados de planejado × realizado, cruzando `tbpmp_item` com dados de produção do OEE (tabelas existentes do sistema)
7. Todas as funções respeitam controle de acesso: `tbusuario.perfil != 'Operador'`
8. Tratamento de erros padronizado com mensagens em pt-BR

  - [ ] Types para filtros: `FiltrosPmp` (pmp_id, semana?, linhaproducao_id?, departamento_id?)

- [ ] Criar `src/services/api/pmp.api.ts` — CRUD Plano (AC: 2, 7)
  - [ ] `listarPlanos()` → `tbpmp_plano` com join para `updated_by` (nome do usuário)
  - [ ] `buscarPlano(pmp_id)` → plano + itens + headcount
  - [ ] `criarPlano(dados)` → INSERT em `tbpmp_plano` com `created_by` do usuário logado
  - [ ] `atualizarPlano(pmp_id, dados)` → UPDATE com `updated_at = NOW()`, `updated_by = usuário logado`

- [ ] Criar funções de Item (AC: 2, 3, 4)
  - [ ] `salvarItem(dados)` →
    - Buscar velocidade: `SELECT velocidade FROM tbvelocidadenominal WHERE linhaproducao_id = :lid AND produto_id = :pid AND deletado = 'N' LIMIT 1`
    - Se não encontrar: retornar `{ warning: 'Velocidade nominal não cadastrada para esta linha/SKU. Horas planejadas não calculadas.' }`
    - Se encontrar: `horas_planejadas = qtd_planejada / velocidade`
    - INSERT/UPDATE em `tbpmp_item`
  - [ ] `removerItem(item_id)` → soft-delete com `deleted_at`, `deleted_by`
  - [ ] `listarItens(pmp_id, filtros?)` → itens com `linha_nome`, `produto_descricao`, indicador `sem_velocidade`

- [ ] Criar funções de Headcount (AC: 5)
  - [ ] `salvarHeadcount(dados)` → INSERT/UPDATE em `tbpmp_headcount`
  - [ ] `removerHeadcount(headcount_id)` → DELETE físico (sem histórico necessário)
  - [ ] `listarHeadcountComAtual(pmp_id)` →
    ```sql
    SELECT h.*,
           l.linhaproducao AS linha_nome,
           (SELECT COUNT(*) FROM tbfuncionario f
            WHERE f.lotacao_id = h.lotacao_id
              AND f.cargo_id = h.cargo_id
              AND f.ativo = true
              AND f.dt_rescisao IS NULL) AS headcount_atual
    FROM tbpmp_headcount h
    JOIN tblinhaproducao l ON l.linhaproducao_id = h.linhaproducao_id
    WHERE h.pmp_id = :pmp_id
    ```
  - [ ] Campo calculado `gap = headcount_desejado - headcount_atual` adicionado no retorno

- [ ] Criar função Comparativo (AC: 6)
  - [ ] `getPmpComparativo(filtros)` → cruzar `tbpmp_item` com dados de produção OEE
  - [ ] Identificar a tabela/view correta de "realizado" no projeto para obter qtd produzida e horas por linha/período
  - [ ] Calcular `aderencia_pct = (qtd_realizada / qtd_planejada) * 100`
  - [ ] Incluir headcount desejado vs atual por linha

- [ ] Funções auxiliares (AC: 4, 8)
  - [ ] `buscarLotacoesDisponiveis()` → `SELECT DISTINCT lotacao_id, lotacao FROM tbfuncionario WHERE ativo = true ORDER BY lotacao`
  - [ ] `buscarCargosPorLotacao(lotacao_id)` → `SELECT DISTINCT cargo_id, cargo FROM tbfuncionario WHERE lotacao_id = :id AND ativo = true ORDER BY cargo`

### Cálculo de horas_planejadas
```typescript
// Buscar velocidade
const { data: vel } = await supabase
  .from('tbvelocidadenominal')
  .select('velocidade')
  .eq('linhaproducao_id', linhaproducao_id)
  .eq('produto_id', produto_id)
  .eq('deletado', 'N')
  .maybeSingle();

const horas_planejadas = vel ? qtd_planejada / vel.velocidade : null;
const sem_velocidade = !vel;
```

### Cálculo de headcount_atual
```sql
-- Executar via RPC ou subquery
SELECT COUNT(*)::INTEGER AS headcount_atual
FROM tbfuncionario
WHERE lotacao_id = $1
  AND cargo_id = $2
  AND ativo = true
  AND dt_rescisao IS NULL
```

### Controle de Acesso

```typescript
const { data: usuario } = await supabase

  .select('perfil')
  .eq('user_id', authUser.id)
  .single();

if (usuario?.perfil === 'Operador') {
  throw new Error('Acesso negado: Operadores não têm acesso ao módulo PMP.');
}
```

# Story 009.001: Migration — Tabelas tbpmp_plano, tbpmp_item, tbpmp_headcount

## Status
Draft

## Story

**Como** desenvolvedor do sistema OEE,
**Eu quero** criar as migrations das tabelas do módulo PMP com estrutura completa e padrão de auditoria do projeto,
**Para que** os dados de planejamento mestre de produção sejam persistidos com integridade referencial e rastreabilidade.

## Acceptance Criteria

1. Tabela `tbpmp_plano` criada com: `pmp_id SERIAL PK`, `ano INTEGER`, `mes INTEGER CHECK(1-12)`, `descricao TEXT`, `ativo BOOLEAN DEFAULT TRUE`, campos de auditoria completos
2. Constraint `UNIQUE(ano, mes)` em `tbpmp_plano` — garante um plano por mês/ano
3. Tabela `tbpmp_item` criada com FKs para `tbpmp_plano(pmp_id)`, `tblinhaproducao(linhaproducao_id)`, `tbproduto(produto_id)`
4. Campos de `tbpmp_item`: `semana INTEGER CHECK(1-5)`, `dt_inicio DATE`, `dt_fim DATE`, `qtd_planejada NUMERIC(15,4) NOT NULL`, `horas_planejadas NUMERIC(10,2)` (nullable — calculado pela aplicação)
5. Constraint `UNIQUE(pmp_id, linhaproducao_id, produto_id, semana)` em `tbpmp_item`
6. Tabela `tbpmp_headcount` criada com FKs para `tbpmp_plano(pmp_id)`, `tblinhaproducao(linhaproducao_id)`
7. Campos de `tbpmp_headcount`: `lotacao_id INTEGER NOT NULL`, `lotacao TEXT`, `cargo_id INTEGER NOT NULL`, `cargo TEXT`, `headcount_desejado INTEGER NOT NULL`
8. Todos os campos de auditoria seguem padrão `tblinhaproducao`: `created_at TIMESTAMP WITHOUT TIME ZONE DEFAULT (NOW() AT TIME ZONE 'America/Fortaleza')`, `created_by INTEGER REFERENCES tbusuario(usuario_id)`, `updated_at TIMESTAMP WITHOUT TIME ZONE`, `updated_by INTEGER REFERENCES tbusuario(usuario_id)`, `deleted_at TIMESTAMP WITHOUT TIME ZONE`, `deleted_by INTEGER REFERENCES tbusuario(usuario_id)`
9. Índices criados para consultas frequentes
10. Migration executada sem erros no Supabase

## Dev Notes

### Database Conventions
- **Referência de padrão:** `tblinhaproducao` — usar exatamente o mesmo padrão de auditoria
- **IDs:** `SERIAL` (não `UUID`, não `IDENTITY`) — padrão do projeto para novas tabelas
- **Timestamps:** `TIMESTAMP WITHOUT TIME ZONE` com `DEFAULT (NOW() AT TIME ZONE 'America/Fortaleza')`
- **FKs de auditoria:** `INTEGER REFERENCES tbusuario(usuario_id)` — NÃO usar UUID
- **`horas_planejadas`:** Manter como campo `NUMERIC(10,2)` simples (nullable) — será populado pela camada de aplicação, NÃO como `GENERATED ALWAYS AS` (pois requer join com outra tabela)

### DDL Completo de Referência

```sql
-- Tabela 1: Cabeçalho do Plano Mensal
CREATE TABLE public.tbpmp_plano (
  pmp_id        SERIAL NOT NULL,
  ano           INTEGER NOT NULL,
  mes           INTEGER NOT NULL,
  descricao     TEXT,
  ativo         BOOLEAN DEFAULT TRUE,
  created_at    TIMESTAMP WITHOUT TIME ZONE DEFAULT (NOW() AT TIME ZONE 'America/Fortaleza'),
  created_by    INTEGER,
  updated_at    TIMESTAMP WITHOUT TIME ZONE,
  updated_by    INTEGER,
  deleted_at    TIMESTAMP WITHOUT TIME ZONE,
  deleted_by    INTEGER,
  CONSTRAINT tbpmp_plano_pkey PRIMARY KEY (pmp_id),
  CONSTRAINT tbpmp_plano_ano_mes_key UNIQUE (ano, mes),
  CONSTRAINT tbpmp_plano_mes_check CHECK (mes BETWEEN 1 AND 12),
  CONSTRAINT tbpmp_plano_created_by_fkey FOREIGN KEY (created_by) REFERENCES tbusuario(usuario_id),
  CONSTRAINT tbpmp_plano_updated_by_fkey FOREIGN KEY (updated_by) REFERENCES tbusuario(usuario_id),
  CONSTRAINT tbpmp_plano_deleted_by_fkey FOREIGN KEY (deleted_by) REFERENCES tbusuario(usuario_id)
) TABLESPACE pg_default;

-- Tabela 2: Itens do Plano
CREATE TABLE public.tbpmp_item (
  item_id           SERIAL NOT NULL,
  pmp_id            INTEGER NOT NULL,
  linhaproducao_id  INTEGER NOT NULL,
  produto_id        INTEGER NOT NULL,
  semana            INTEGER NOT NULL,
  dt_inicio         DATE NOT NULL,
  dt_fim            DATE NOT NULL,
  qtd_planejada     NUMERIC(15,4) NOT NULL,
  horas_planejadas  NUMERIC(10,2),
  created_at        TIMESTAMP WITHOUT TIME ZONE DEFAULT (NOW() AT TIME ZONE 'America/Fortaleza'),
  created_by        INTEGER,
  updated_at        TIMESTAMP WITHOUT TIME ZONE,
  updated_by        INTEGER,
  deleted_at        TIMESTAMP WITHOUT TIME ZONE,
  deleted_by        INTEGER,
  CONSTRAINT tbpmp_item_pkey PRIMARY KEY (item_id),
  CONSTRAINT tbpmp_item_unique UNIQUE (pmp_id, linhaproducao_id, produto_id, semana),
  CONSTRAINT tbpmp_item_semana_check CHECK (semana BETWEEN 1 AND 5),
  CONSTRAINT tbpmp_item_pmp_fkey FOREIGN KEY (pmp_id) REFERENCES tbpmp_plano(pmp_id),
  CONSTRAINT tbpmp_item_linha_fkey FOREIGN KEY (linhaproducao_id) REFERENCES tblinhaproducao(linhaproducao_id),
  CONSTRAINT tbpmp_item_produto_fkey FOREIGN KEY (produto_id) REFERENCES tbproduto(produto_id),
  CONSTRAINT tbpmp_item_created_by_fkey FOREIGN KEY (created_by) REFERENCES tbusuario(usuario_id),
  CONSTRAINT tbpmp_item_updated_by_fkey FOREIGN KEY (updated_by) REFERENCES tbusuario(usuario_id),
  CONSTRAINT tbpmp_item_deleted_by_fkey FOREIGN KEY (deleted_by) REFERENCES tbusuario(usuario_id)
) TABLESPACE pg_default;

-- Tabela 3: Headcount Planejado por Linha
CREATE TABLE public.tbpmp_headcount (
  headcount_id        SERIAL NOT NULL,
  pmp_id              INTEGER NOT NULL,
  linhaproducao_id    INTEGER NOT NULL,
  lotacao_id          INTEGER NOT NULL,
  lotacao             TEXT,
  cargo_id            INTEGER NOT NULL,
  cargo               TEXT,
  headcount_desejado  INTEGER NOT NULL,
  created_at          TIMESTAMP WITHOUT TIME ZONE DEFAULT (NOW() AT TIME ZONE 'America/Fortaleza'),
  created_by          INTEGER,
  updated_at          TIMESTAMP WITHOUT TIME ZONE,
  updated_by          INTEGER,
  CONSTRAINT tbpmp_headcount_pkey PRIMARY KEY (headcount_id),
  CONSTRAINT tbpmp_headcount_pmp_fkey FOREIGN KEY (pmp_id) REFERENCES tbpmp_plano(pmp_id),
  CONSTRAINT tbpmp_headcount_linha_fkey FOREIGN KEY (linhaproducao_id) REFERENCES tblinhaproducao(linhaproducao_id),
  CONSTRAINT tbpmp_headcount_created_by_fkey FOREIGN KEY (created_by) REFERENCES tbusuario(usuario_id),
  CONSTRAINT tbpmp_headcount_updated_by_fkey FOREIGN KEY (updated_by) REFERENCES tbusuario(usuario_id)
) TABLESPACE pg_default;

-- Índices
CREATE INDEX idx_tbpmp_plano_ano_mes ON public.tbpmp_plano(ano, mes) WHERE ativo = TRUE;
CREATE INDEX idx_tbpmp_item_pmp ON public.tbpmp_item(pmp_id);
CREATE INDEX idx_tbpmp_item_linha ON public.tbpmp_item(linhaproducao_id, pmp_id);
CREATE INDEX idx_tbpmp_item_semana ON public.tbpmp_item(pmp_id, semana);
CREATE INDEX idx_tbpmp_headcount_linha ON public.tbpmp_headcount(pmp_id, linhaproducao_id);
```

### Atenção: tbvelocidadenominal
A tabela `tbvelocidadenominal` possui colunas `updated_at`/`updated_by` e `deleted_at`/`deleted_by` com nomes trocados na DDL (integer onde deveria ser timestamp). Ao usar essa tabela, mapear apenas os campos seguros: `velocidade_id`, `linhaproducao_id`, `produto_id`, `velocidade`, `deletado`. Não replicar esse padrão nas novas tabelas.

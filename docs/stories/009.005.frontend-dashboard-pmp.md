# Story 009.005: Frontend — Dashboard PMP (Planejado × Realizado)

## Status
Draft

## Story

**Como** Diretor ou Gestor PCP,
**Eu quero** visualizar o plano de produção comparado ao executado com indicadores de aderência,
**Para que** eu possa avaliar o desempenho do mês e tomar decisões de ajuste na reunião semanal.

## Acceptance Criteria

1. Rota `/pmp/dashboard` acessível para `perfil != 'Operador'`
2. Filtros no topo: Mês/Ano (select do plano), Semana (1-5 ou Todas), Linha (todas ou específica), Departamento
3. KPIs no topo (cards):
   - **Aderência Geral do Mês (%)**: média ponderada de todas as linhas
   - **Total Horas Planejadas vs Realizadas**: "1.234,5 h / 1.180,2 h"
   - **Linhas com Déficit de Headcount**: número de linhas onde `headcount_atual < headcount_desejado`
   - **Linhas Abaixo de 85% de Aderência**: contador
4. Tabela comparativa com colunas: Linha | SKU | Semana | Qtd Planejada | Qtd Realizada | % Aderência | Horas Planejadas | Horas Realizadas | HC Desejado | HC Atual | Gap HC
5. Sinalização de % Aderência:
   - ≥ 95%: verde (badge)
   - 85-94%: amarelo (badge)
   - < 85%: vermelho (badge)
6. Sinalização de Gap HC:
   - ≥ 0: verde
   - < 0: vermelho
7. Linhas sem produção realizada no período exibem "—" em Qtd Realizada e Horas Realizadas (não ocultar)
8. Tabela ordenável por qualquer coluna
9. Todos os números em formatação pt-BR

### Cálculo de Aderência
```typescript
// Por linha: aderencia = (qtd_realizada / qtd_planejada) * 100
// Geral: media ponderada pelo planejado
const aderenciaGeral = totalRealizado > 0
  ? (totalRealizado / totalPlanejado) * 100
  : 0;

// Classificação de cor
const getCorAderencia = (pct: number) => {
  if (pct >= 95) return 'green';
  if (pct >= 85) return 'yellow';
  return 'red';
};
```

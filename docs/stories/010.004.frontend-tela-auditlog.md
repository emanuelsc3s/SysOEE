# Story 010.004: Frontend — Tela de Consulta do Audit Log

## Status
Draft

## Story

**Como** supervisor ou auditor de qualidade,
**Eu quero** consultar o histórico de alterações de qualquer registro do sistema,
**Para que** eu possa rastrear quem alterou o quê e quando, respondendo a questionamentos em auditorias ANVISA.

## Acceptance Criteria

1. Rota `/auditoria` criada, acessível somente para perfis `Supervisor`, `Encarregado`, `Gestor` e `Admin`
2. Página exibe tabela com as colunas: **Data/Hora**, **Tabela**, **Operação** (badge colorido), **Campo**, **Antes**, **Depois**, **Usuário**, **IP**
3. Filtros disponíveis:
   - **Tabela** — select com lista das tabelas auditadas (multi-select)
   - **Operação** — select: INSERT / UPDATE / DELETE (multi-select)
   - **Usuário** — select com lista de usuários (busca por nome)
   - **Data início** e **Data fim** — date pickers
   - **Busca livre** — campo de texto que filtra por `registro_id` ou conteúdo de `valor_anterior`/`valor_novo`
4. Paginação com 50 registros por página, ordenação por `timestamp DESC` (mais recente primeiro)
5. Operação **INSERT** exibida com badge **verde**; **UPDATE** com badge **amarelo**; **DELETE** com badge **vermelho**
6. Coluna **Antes** e **Depois**: se o valor for um JSON (INSERT/DELETE completo), exibir botão "Ver detalhes" que abre um modal com o JSON formatado (pretty-print)
7. Coluna **Usuário**: exibir `nome` do usuário (JOIN com `tbusuario`); se `usuario_id = NULL`, exibir "Sistema"
8. Coluna **IP**: exibir o endereço IP; se NULL, exibir "—"
9. Botão **Exportar CSV** que exporta os registros filtrados atualmente (sem limite de 50)
10. Performance: query deve retornar em menos de 2 segundos para filtros típicos (índices já existem em `tbauditlog`)
11. Perfil `Operador` não deve ter acesso à rota — redirecionar para página de acesso negado

### Query base de referência

```typescript
const { data, count } = await supabase
  .from('vw_auditlog')
  .select('*', { count: 'exact' })
  .in('tabela', filtros.tabelas ?? tabelasDisponiveis)
  .in('operacao', filtros.operacoes ?? ['INSERT', 'UPDATE', 'DELETE'])
  .gte('timestamp', filtros.data_inicio)
  .lte('timestamp', filtros.data_fim)
  .order('timestamp', { ascending: false })
  .range(pagina * 50, (pagina + 1) * 50 - 1);
```

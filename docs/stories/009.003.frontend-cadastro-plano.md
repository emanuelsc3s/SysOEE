# Story 009.003: Frontend — Cadastro e Revisão de Plano Mensal

## Status
Draft

## Story

**Como** Gestor PCP,
**Eu quero** criar e revisar o plano de produção mensal por linha, SKU e semana,
**Para que** as metas de produção do mês estejam formalizadas e atualizadas semanalmente.

## Acceptance Criteria

1. Rota `/pmp` acessível para `perfil != 'Operador'`; redirecionar Operadores para tela de acesso negado
2. Tela de lista de planos exibe: Mês/Ano, Descrição, Data da última revisão, Usuário da última revisão, botão "Abrir"
3. Botão "Novo Plano" → modal com campos: Mês (select 1-12), Ano (input numérico), Descrição (texto opcional)
4. Sistema impede criação de plano duplicado para o mesmo mês/ano (erro amigável em pt-BR)
5. Tela de edição do plano exibe header com: mês/ano, descrição e banner "Última revisão: [DD/MM/AAAA] por [Nome]"
6. Tabela de itens do plano com colunas: Linha | SKU | Semana | Período | Qtd Planejada | Horas Planejadas | Ações
7. Botão "Adicionar Item" → formulário com:
   - Select de Linha (`tblinhaproducao`)
   - Select de SKU/Produto (`tbproduto`) — filtrável por texto
   - Select de Semana (1, 2, 3, 4, 5)
   - Inputs de Data Início e Data Fim
   - Input de Qtd Planejada (formatação pt-BR: separador de milhar ponto, decimal vírgula)
   - Campo "Horas Planejadas" — exibido automaticamente após selecionar Linha + SKU + Qtd; readonly
8. Aviso visual amarelo quando velocidade nominal não está cadastrada para o par Linha/SKU selecionado, informando que horas não serão calculadas
9. Ao salvar item: `updated_at = NOW()` e `updated_by = usuário logado` gravados no plano
10. Banner "Última revisão" atualizado imediatamente após salvar qualquer item
11. Itens podem ser editados e removidos (soft-delete com `deleted_at`, `deleted_by`)
12. Sem fluxo de aprovação — qualquer salvamento entra em vigor imediatamente

### Cálculo de Horas em Tempo Real (preview)
```typescript
// Ao selecionar linha + SKU + digitar qtd
const calcularHorasPreview = async (linhaId: number, produtoId: number, qtd: number) => {
  const { data: vel } = await supabase
    .from('tbvelocidadenominal')
    .select('velocidade')
    .eq('linhaproducao_id', linhaId)
    .eq('produto_id', produtoId)
    .eq('deletado', 'N')
    .maybeSingle();

  if (!vel) return { horas: null, semVelocidade: true };
  return { horas: qtd / vel.velocidade, semVelocidade: false };
};
```
